# Whack-a-Goblin Game

[![Build and Deploy](https://github.com/AGAFONOV73/ahj-homeworks/actions/workflows/deploy.yml/badge.svg)](https://github.com/AGAFONOV73/ahj-homeworks/actions)
[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://agafonov73.github.io/ahj-homeworks/)

## Описание
Игра "Ударь гоблина". Гоблин появляется в случайных клетках игрового поля 4x4 на 1 секунду. 
Цель - набрать как можно больше очков. 5 пропущенных гоблинов - игра окончена.

## Особенности
- Кастомный курсор в виде молотка
- Таймер реакции: 1 секунда на удар
- Счетчик пропусков (максимум 5)
- Автоматический деплой на GitHub Pages

## Технологии
- HTML5/CSS3
- JavaScript (ES6+)
- Webpack
- Yarn
- GitHub Actions
